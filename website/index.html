How it works

Three steps to peace of mind

1

Install globally

git clone https://github.com/nizernoj/Checkpoint.git
cd Checkpoint && ./bin/install-global.sh
2

Run in any project

cd /your/project
backup-now
3

Done. Forever.

Project registered. Hourly backups enabled. Go build something.
