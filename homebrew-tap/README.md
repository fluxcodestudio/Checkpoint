# Homebrew Tap for Checkpoint

Install [Checkpoint](https://checkpoint.fluxcode.studio) via Homebrew.

## Usage

```bash
brew tap fluxcodestudio/checkpoint
brew install checkpoint
```

Then run `backup-now` in any project directory.
