## Phases

### Phase 10: Dashboard Settings Integration

**Goal**: Add settings menu and wizard launcher to dashboard for unified UX
**Depends on**: v1.1 complete
**Plans**: 2 plans

Plans:
- [x] 10-01: Add "Settings" menu to dashboard with key options
- [x] 10-02: Add "Launch config wizard" quick action

**Details:**

Settings submenu features:
- View current settings (alerts, notifications, quiet hours, cloud)
- Edit alert thresholds (warning/error hours)
- Edit quiet hours (HH-HH format with overnight support)
- Edit notification sound preferences
- Launch full setup wizard

Implementation:
- Both dialog (TUI) and text fallback versions
- Quick edits update runtime environment variables
- Wizard saves changes permanently to config file
